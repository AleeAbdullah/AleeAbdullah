# Quick Reference: Replace These Placeholders

Before pushing to GitHub, replace these placeholders in `README.md`:

## Required Replacements:

1. **YOUR_USERNAME** - Replace with your actual GitHub username (appears in multiple places)
   - Line 1-3: Badge URLs
   - Line 10: Socialify image
   - Line 25: GitHub profile trophy
   - Line 28: Activity graph

2. **YOUR_LINKEDIN_USERNAME** - Replace with your LinkedIn profile username
   - Line 15: LinkedIn link
   - Example: If your LinkedIn is `https://www.linkedin.com/in/ali-abdullah`, replace with `ali-abdullah`

## Optional Replacements:

3. **Spotify Integration** (if you want it)
   - Uncomment the Spotify sections (lines 55-61)
   - Replace `YOUR_SPOTIFY_USER_ID` with your Spotify user ID
   - You can find this in your Spotify profile URL

## Quick Find & Replace:

Use your editor's find & replace feature:
- Find: `YOUR_USERNAME`
- Replace: `your-actual-github-username`
- Replace all

- Find: `YOUR_LINKEDIN_USERNAME`
- Replace: `your-linkedin-username`
- Replace all

## After Replacing:

1. Save the file
2. Commit and push to GitHub
3. The GitHub Actions workflows will automatically update the activity and stats sections
